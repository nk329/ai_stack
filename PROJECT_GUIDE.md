# AI 자동매매 프로그램 - PROJECT GUIDE

## 프로젝트 개요

- **목표**: AI를 활용한 국내주식 + 미국주식 + 암호화폐 자동매매로 수익 창출
- **초기 자본**: 100,000원 (이 금액을 모두 잃으면 파산)
- **핵심 원칙**: 자본 보존 최우선, 리스크 관리 철저

---

## 투자 철학 (참고 서적 및 투자 대가)

| 원칙 | 내용 | 출처 |
|------|------|------|
| 자본 보존 | "첫째도 잃지 마라, 둘째도 잃지 마라" | 워렌 버핏 |
| 추세 추종 | 시장의 방향성을 따라가라 | 제시 리버모어 |
| 손절 원칙 | 손실은 빠르게(-3%), 수익은 천천히 | 마켓 위저드 |
| 분산 투자 | 한 자산군에 몰빵 금지 | 벤저민 그레이엄 |
| 백테스트 | 실전 전 반드시 과거 데이터로 검증 | 퀀트 투자 바이블 |
| 켈리 공식 | 승률과 손익비 기반 포지션 사이징 | 켈리 기준 |
| 감성 분석 | 뉴스/SNS 심리 지표 활용 | AI 트레이딩 |

## 기술 스택

### 언어 & 환경
- **Python 3.11+**
- **가상환경**: venv

### 증권/거래소 API
| 시장 | API | 비고 |
|------|-----|------|
| 국내주식 | 한국투자증권 KIS Developers | 무료, REST + WebSocket |
| 미국주식 | 한국투자증권 KIS Developers | 동일 API로 해외주식 지원 |
| 암호화폐 | 업비트(Upbit) Open API | 무료, KRW 마켓 |

### 데이터 수집
- `pykrx` - 국내 주식 시세
- `FinanceDataReader` - 국내/해외 주가
- `yfinance` - 미국 주식 데이터
- `pyupbit` - 업비트 암호화폐

### AI / 분석
- `openai` - GPT-4 뉴스 감성분석
- `scikit-learn` - 머신러닝 모델
- `lightgbm` - 예측 모델
- `ta` (Technical Analysis) - 기술적 지표

### 인프라
- `SQLite` - 거래 기록 DB
- `APScheduler` - 자동 스케줄링
- `python-telegram-bot` - 텔레그램 알림
- `pandas`, `numpy` - 데이터 처리
- `matplotlib`, `plotly` - 시각화

## 리스크 관리 규칙

```
초기 자본: 100,000원
최대 단일 종목 투자: 자본의 20% (20,000원)
손절 기준: -3% (종목당)
익절 기준: +5% ~ +10% (전략별 상이)
일일 최대 손실 한도: 자본의 5% (5,000원)
최대 낙폭(MDD) 한도: 30% → 초과 시 전체 매매 중지
```

## API 발급 가이드

### 1. 한국투자증권 KIS Developers
1. https://apiportal.koreainvestment.com 접속
2. 회원가입 후 앱 등록
3. `APP_KEY`, `APP_SECRET` 발급
4. 모의투자 계좌 개설 (실전 전 필수)

### 2. 업비트 Open API
1. https://upbit.com 로그인
2. 마이페이지 → Open API 관리
3. `ACCESS_KEY`, `SECRET_KEY` 발급
4. 입출금/거래 권한 설정

### 3. OpenAI API (감성분석용)
1. https://platform.openai.com 접속
2. API Keys 발급
3. GPT-4o-mini 모델 사용 (비용 절약)

### 4. 텔레그램 Bot
1. @BotFather 에서 봇 생성
2. `BOT_TOKEN` 발급
3. 본인 `CHAT_ID` 확인

## AI 스크리너 점수 체계 (Phase A)

| 항목 | 배점 | 기준 |
|------|------|------|
| 모멘텀 | 30점 | 7일/30일 수익률, 이동평균 정배열 |
| 기술지표 | 30점 | RSI(30~50 최적), 볼린저밴드 하단, MACD 반등 |
| 거래량 | 20점 | 20일 평균 대비 비율 (2배+ 우선) |
| 안정성 | 20점 | 일일 변동성 (낮을수록 좋음) |
| **합계** | **100점** | **40점 미만 진입 제외** |

### 포트폴리오 구성 원칙
- 코인 최대 3개, 주식 최대 3개 (동시 보유)
- 단일 종목 최대 비중 30%
- 점수 하락 시 자동 교체 (교체 임계값: 15점 차이)
- 자본금 90% 투자 + 10% 현금 보유
